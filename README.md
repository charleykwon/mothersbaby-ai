# 🤱 육아 컴패니언 AI - Phase 1 MVP

**맞춤형 모유수유 가이드 챗봇**

25년 경력 IBCLC(국제모유수유전문가) 지식을 24/7 제공하는 AI 챗봇입니다.

---

## ✨ 주요 기능

### 🎯 컨텍스트 우선 수집
- 질문 전 아기 월령, 주제, 상황 파악
- 맞춤형 답변 제공
- 긴급 상황 자동 분류

### 📚 15개 핵심 답변
- 젖물림/자세 (3개)
- 젖양 걱정 (4개)
- 가슴 통증/염증 (4개)
- 아기 체중/수유량 (4개)
- 복직/유축 (4개)
- 기타 (3개)

### 🚨 긴급 상황 대응
- 38도 이상 고열
- 젖몸살 초기 대응
- 의료진 상담 기준 명시

---

## 🚀 기술 스택

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Hosting**: Vercel (무료)
- **Design**: 반응형, 모바일 최적화
- **특징**: 단일 HTML 파일 (Zero Dependency)

---

## 📱 데모

**라이브 데모**: [https://mothersbaby-ai.vercel.app](배포 후 URL)

### 사용 흐름
```
Step 1: 아기 월령 선택 (신생아 / 1-3개월 / 4-6개월 / 6개월+)
  ↓
Step 2: 주제 선택 (젖양 걱정 / 가슴 통증 등 6가지)
  ↓
Step 3: 구체적 상황 선택
  ↓
Step 4: 맞춤형 답변 제공
```
